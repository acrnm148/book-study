### 1. Repository 인터페이스
- `spring-data-commons` 모듈에 포함되어 있음.
- spring data repository 모듈은 Repository 인터페이스를 사용해서 데이터 소스 접근을 추상화
- Repository 인터페이스는 도메인 클래스 + 식별자 타입을 필요로 한다.
  - 도메인 클래스 = DB에 저장되어 관리되는 비즈니스 엔티티 클래스
